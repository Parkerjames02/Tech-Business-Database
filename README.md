## 📝 Simple Note Taking App
This is a simple database that tracks data for a simulated tech business. It keeps track of information like store locations 
along with store owner, the employees that work at each store including their name, position, and join date, the customers that 
make purchases at these stores including their name, the ID number of the store that they bought from, the employee that helped 
them with their purchase, as well as the total cost of the products that they purchased and the purchase date.
## ✨ Technologies
- SQL
- MySQL Workbench
## 🚀 Features
- Store Data
- Employee Data
- Customer Data
- Purchase Data
## 📌 The Reason
I made this database to meet the requirements of my final project in my Introduction to Databases class during my time in
college. This database project is meant to encapsulate everything that I had learned during the length of the course. There is
a total of four different tables that have a relationship in some way with another table with the purchase information table
being connnected to the other three tables.
## 📖 How To Use
Every query that I used to create the database schema, tables, and data is within the "tech_business_database.sql". The 
different select statements were a requirement for the assignment and can be used to view different types of data from the 
tables. I did everything in MySQL Workbench so using the same application or a similar application is recommended.
